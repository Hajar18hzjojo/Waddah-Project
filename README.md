# 🏠 Project Waddah (وضاح) - Smart Energy Monitoring System

**Waddah** is an integrated smart system designed to monitor and analyze home energy consumption using **Non-Intrusive Load Monitoring (NILM)** and **Edge AI** technology. It provides real-time data, cost estimation based on Saudi electricity tariffs, and an early warning system for electrical faults—all while ensuring total user privacy through local data processing.

## 🚀 Key Features
* **NILM Technology**: Identifies individual appliances' "power fingerprints" from a single monitoring point.
* **Edge AI (Privacy First)**: All data analysis happens locally on the ESP32; no cloud or internet connection is required.
* **KSA Electricity Tariff Integration**: Real-time cost calculation based on Saudi billing slices.
* **Early Fault Detection**: Detects overloads and abnormal fluctuations to protect the home.

---

## 🛠️ Technical Overview
* **Microcontroller**: ESP32.
* **Sensors**: SCT-013 (Non-invasive clamp) & ADS1115.
* **Software**: Developed using **React Native** for the mobile application.
* **Hardware**: Professional **DIN rail** mounting inside the main electrical panel.
